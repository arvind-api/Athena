WHAT THE DASHBOARD MUST LOOK LIKE

Full dark theme. Deep space black background (#0a0a0f).
Neon accent colors:
  - Cyan (#00ffff) for active symbols and edges
  - Purple (#a855f7) for emerging concepts
  - Green (#00ff88) for stored memories
  - Orange (#ff6b35) for contradictions and warnings
  - White (#ffffff) for text

Layout — three panels side by side:

LEFT PANEL — "MEMORY STREAM" (30% width)
  - Scrolling live feed of every interaction
  - Each entry shows: question, answer snippet, similarity score if matched
  - New entries slide in from the top with a fade animation
  - Color coded: green glow if related memory found, purple if concept emerging
  - Shows: "Memory #N | similarity: 0.82 | keywords: [recursion, function]"

CENTER PANEL — "KNOWLEDGE GRAPH" (40% width)
  - D3.js force-directed graph
  - Nodes = symbols (RECURSION, REFERS, etc.)
  - Node size scales with frequency — more seen = bigger node
  - Node color: cyan if active, orange if contradiction candidate
  - Edges glow cyan when two symbols are connected
  - When a new symbol appears: node materializes with an expanding ring animation
  - When symbols combine: two nodes merge with a particle burst effect
  - Nodes have labels. Hovering shows: name, frequency, keywords, first seen
  - The graph slowly rotates when idle — it looks alive
  - Background: subtle grid of dots (like a circuit board)

RIGHT PANEL — "COGNITIVE STATUS" (30% width)
  - Top section: ATHENA system stats
      - Total memories stored
      - Active symbols count
      - Graph edges count
      - Session uptime
  - Middle section: live "THOUGHT STREAM"
      - Shows the last 3 compressed summaries as they come in
      - Each one types in character by character (typewriter effect)
  - Bottom section: AUDIT STATUS
      - Live audit report from last session end
      - Contradiction warnings glow orange
      - Combinable pairs highlighted purple

HEADER — full width
  - "ATHENA" in large monospace font, glowing cyan
  - Subtitle: "Autonomous Thought Heuristic Engine for Neural Abstraction"
  - Top right: connection status dot (green = connected, red = disconnected)
  - Top right: "THINKING..." pulse animation when inference is running
